# 🎬 Movie Explorer App

## 📌 Description

The Movie Explorer App is a web application that allows users to browse and explore movies using real-time data from the TMDB (The Movie Database) API. Users can search for movies, filter them based on ratings, and sort them by popularity or release date.

This project is built as part of a learning exercise to demonstrate the use of JavaScript, API integration, and dynamic UI rendering.

---

## 🚀 Features (Planned)

* 🔍 Search movies by title
* 🎯 Filter movies by rating
* 📊 Sort movies by popularity and release date
* ❤️ Add movies to favorites
* 🌙 Dark/Light mode toggle

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* TMDB API

---

## 🌐 API Used

TMDB (The Movie Database)
https://www.themoviedb.org/

---

## ⚙️ Setup Instructions

1. Clone the repository
2. Open the project folder
3. Add your TMDB API key in the JavaScript file
4. Open `index.html` in your browser
